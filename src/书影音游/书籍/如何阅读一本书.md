# 如何阅读一本书

## 阅读的层次

### 阅读的活力与艺术

- 前言

	- 为想把读书的主要目的当做是增进理解能力的人而写

	- 现在的媒体以压倒性的泛滥资讯阻碍了我们的理解力

		- 直接将包装后的观点装进自己脑海中，不需要自己思考

- 主动阅读

	- 阅读越主动，效果越好。头脑昏睡时很难阅读

	- 想理解多少完全看他在阅读时多么主动以及投入多少心思与阅读技巧

- 阅读目标

	- 只有一种方式是真正的阅读

		- 没有借助外力，你就是要读这本书

		- 从只有模糊的概念到更清楚的理解为止

			- 从粗浅了解推进到深入的体会，就像是破茧而出

	- 为获得资讯而读

		- 认知和内容对等

		- 学不到什么内容，作为一个消遣和娱乐

		- 可以尝试理解

			- 搞清楚这个资讯背后的认知

				- 为什么会发生
与其他事有什么关联
有什么类似情况
同类的差异在哪儿
...

	- 为求得理解而读

		- 内容高于认知

		- 从比我们更牛逼的人身上学习

		- 只要努力运用技巧在有益的读物上，每个人都能 读的更好，学的更多

- 阅读就是学习

	- 除了知道作者所说的话，还要懂得他为什么这样说

	- 读得多!=读的好

		- 读不懂相当于没读

		- 初学者的无知在于未学，学者的无知在于学后

- 阅读是跟一位缺席的老师学习

	- 问一本书一个问题，必须自己回答这个问题

		- 只有做了思考与分析之后，才能在书本上找到答案

	- 懂得让书本来教导我们，是本书最重要的目的

###  阅读的层次

- 前言

	- 阅读的目标决定阅读的方式

	- 层次不同于种类，层次是高层包含低层，而且渐进

- 基础阅读

	- 了解 这个句子在说什么

	- 大部分速读课程着眼于这个层次

- 检视阅读

	- 规定时间内做系统化略读

	- 了解 这本书在说什么

		- 这本书架构如何
这本书包含那些部分
这是哪一类的书
...

- 分析阅读，本书重点！

	- 在无限时间里，最好也最完整的阅读

		- 用于寻求理解

	- 要对所读的东西提出许多系统的问题

	- 紧抓一本书，读到这本书成为他自己为止

		- 分析阅读就是要咀嚼与消化一本书

- 主题阅读

	- 读多本书，列举相关之处，提出一个多本书都谈到的主题

### 基础阅读

- 学习阅读的阶段

	- 1.准备工作

		- 身体发育良好

			- 幼儿园阶段

	- 2.读一些简单读物

		- 一年级阶段

	- 3.快速建立字汇能力

		- 理解上下文

			- 四年级阶段

	- 4.精练增进前面所学技巧

		- 初中阶段

### 检视阅读

- 有系统的略读或粗读

	- 为什么

		- 不知道自己想不想读这本书

		- 不知道这本书是否值得做分析阅读

		- 想要发掘所有的东西，但时间有限

	- 如何做

		- 1.先看书名页，有序先看序

		- 2.研究目录页

			- 对这本书的基础架构做概括性理解

		- 3.检阅索引

			- 快速评估本书涵盖了哪些议题范围

			- 书籍种类、作者、重要词汇

		- 4.出版者介绍

		- 前四步判断是否还想读下去，如果想，下面就是真正略读这本书

		- 5.挑几个看来跟主题息息相关的篇章来看

			- 看摘要说明

		- 6.东翻西翻，念一两段或连读几页，但不要太多

			- 这样把全书翻一遍

				- 寻找主要论点，留意主题的基本脉动

		- 确定本书该归为那一个种类
是否包含你还想继续挖掘的内容
是否值得再继续投下时间与注意

- 粗浅的阅读

	- 面对一本难读的书，从头到尾先读一遍，碰到不懂的不要停下查询或思索

	- 不要被看不懂的东西阻挠导致泄气

	- 必须先从头到尾读一遍

	- 不要碰到困难就停住，如果碰到困难停住，最后可能对这本书真的就一无所知了

		- 困难破坏了阅读的快乐

- 阅读的速度

	- 应该依照读物的性质和复杂程度，用不同的速度去阅读

	- 只需要快速阅读的书用很慢的速度是浪费时间

	- 无关紧要的部分快速阅读、困难的地方慢慢阅读

- 理解的问题

	- 一个优秀的读者要读的主动、专心

	- 专心不等于理解力

	- 能够正确的回答许多进一步的问题，才表示有更高一层的理解力

	- 分析阅读，是想要理解一本书的基本操作

- 摘要

	- 略读是准备了解一本书的架构

	- 粗浅阅读是想要了解全书内容的第一个必要步骤

	- 阅读的本质是一种活动

### 如何做一个自我要求的读者

- 如果你想获得成长，就得保持主动、清醒并努力的阅读

	- 把一本好书当做镇静剂，是一种极度浪费

- 如何做

	- 阅读时要提出问题，阅读过程中，必须去尝试回答的问题

	- 四个主要问题

		- 1.这本书在谈什么

			- 作者如何依次发展这个主题

			- 如何逐步从核心主题分解出从属的关键议题来

		- 2.作者细部说了什么，怎么说的

			- 找出主要的想法、声明与论点

		- 3.这本书说的有道理吗？全部有道理，还是部分有道理

			- 判断这本书是否有道理，必须先了解整本书在说什么

			- 光是知道作者的想法是不够的

		- 4.这本书跟你有什么关系

			- 读这本书有什么意义

			- 这本书的咨询有必要了解吗

			- 找出更深的含义或建议，获得更多的启示

	- 超越基础阅读的阅读层次，核心就在要努力提出问题。这是有自我要求的阅读者和没有自我要求的阅读者有天壤之别的原因。提不出问题，就得不到答案。

- 如何让一本书真正属于你自己

	- 读出言外之意，写出来

	- 做笔记是不可或缺的

		- 让你保持清醒

		- 会有思考

			- 如果一个人说他知道自己在想什么，却说不出来，通常是他不知道自己在想什么

		- 将感想写出来，能帮助你记住作者的思想

	- 如何做

		- 画底线

			- 重点、有感悟的地方画线

		- 双底线

			- 重中之重的地方双底线

		- 画符号

			- 一段都很重要就用符号标记

		- 空白处编号

		- 空白处记下其他的页码

			- 相同论点作比较，可以让全书的想法统筹起来

		- 在书页的空白处做笔记

			- 某些问题或答案记下来

				- 帮助回想

			- 复杂论点简化说明在空白处

			- 记下全书主要论点的发展顺序

				- 书中最后一页可以做个人索引页

					- 做完之后试着将全书大纲、整体架构写出来

						- 测量是否了解了全书

			- 书前空白最好用来记载你的思想

	- 做笔记的方法

		- 结构笔记

			- 做这些笔记最好的地方是目录页或书名页

			- 针对检视阅读

			- 如何做

				- 这是本什么样的书

				- 这本书在谈什么

				- 作者借着怎样的整体架构来发展或陈述他对这个主题的理解

			- 结构笔记的重点是全书的架构，而不是内容或细节

		- 概念笔记

			- 针对分析阅读

			- 关于这本书准确性和意义

			- 作者的观点

		- 辩证笔记

			- 针对一场讨论情境的笔记

			- 由许多作者共同参与

			- 从很多本书摘要出来

			- 单一主题，把所有相关陈述和疑问顺序而列

	- 培养阅读的习惯

		- 要养成习惯，除了不断运作和练习，别无他法

			- 谈到一个有技术的人时，并不是在说他知道如何去做那件事，而是他已经养成去做那件事的习惯了

		- 培养习惯要按规则去练习

		- 照着规则练习技巧或手艺，就可以培养出习惯

		- 规则

			- 许多个动作/规则连在一起，才能学会一项技术，比如滑雪

			- 为了忘掉这些单一的动作，一开始必须先分别学会每一个单一的动作

				- 只有这样才能将所有动作连接起来，变成一个滑雪高手

			- 一开始的复杂度是不足为惧的

				- 只有精通单一动作，才能完成整体行动

			- 动作渐渐融为一体时，学习者便能将注意力转移到目标上

## 分析阅读

### 第一阶段

- 一本书的分类

	- 书籍分类的重要性

		- 一定要知道自己读的是哪一类书，最好早在你开始阅读之前就先知道

		- 一本论说性的书主要目的是传达知识

			- 主要内容是由一些观点、理论、假设、推断所组成，并且这些主张是有一定根据和道理的

		- 分析阅读特别适合来阅读理论性的书

	- 从一本书的书名能学到什么

		- 只有少而又少的人用心读过书名，想过其中的含义

		- 忽略书名的原因之一是因为将书做分类是没有必要的

		- 在心中要有一个分类的标准，否则再清楚的书名，再详尽的目录、前言，对你都没帮助

		- 书名有时会让书籍分类容易一些

		- 光是将书籍分类到某个种类是不够的

			- 一定要知道这个种类的书是在谈些什么

				- 心中有一个分类标准，才能做明智的判断

				- 在不同书籍间找出区别，定出合理的分类。比如：

					- 小说类

						- 人文类

						- 玄幻类

						- 子主题 3

					- 论说类

						- 历史类

						- 哲学类

						- 科学类

						- 数学类

	- 实用性vs理论性作品

		- 适用于论说性作品的分类原则

		- 实用性

			- 与某种有效的做法有关。不管是立即或长程的功效

			- 关注哪些知识能帮忙解决人生问题

			- 教你“如何去做”

				- 如何生活
			该做什么
			不该做什么
			做了有什么奖赏
			不做有什么惩罚

					- 无论是否同意他的结论，都得认定这是一本实用的书

		- 理论性

			- 去明白或了解某些事

			- 坚持为知识而知识

			- 教你“这是什么”

				- 研究报告

				- 数据分析

			- 理论性作品的分类

				- 历史

					- 本质是口述故事

						- 在描述历史时，通常会带有个人色彩

					- 发生在特定时间、特定地点的事

				- 科学

					- 定律或通则

						- 所有情况或大多情况中，事情是如何发生的

					- 实验为基础，精确的观察和研究

					- 一般人很难查证

				- 哲学

					- 追求一般真理

					- 利用人类共通经验来思考

				- 一本超乎你日常、例行、正常生活经验内容的书，一般是科学的书。否则是哲学的书

		- 知识与行动的区别

			- 明智的行动来自知识

			- 知识可以指导人类行为

			- 知识变成实用

				- 要知道是怎么回事，进而明白：如果我们想做什么，应该怎么利用它

		- 一本书不同段落会有不同的分类

		- 为什么要分类

			- 不同类型的书传达的知识不同，指导我们的方式也不同

			- 学习以适当的态度来阅读不同的书

- 透视一本书

	- 每本书都有一套自己的骨架，你的责任就是找出这个骨架

		- 整体性与组织架构

	- 用一个单一的句子或最多几句话来叙述整本书的内容

		- 找出一本书在干什么
发现这本书的主题或重点

		- 主要情节牵扯着所有事情连接在一起

			- 情节是主干，其他都是插曲

			- 同样的情节，一位作家写出来的是好故事或坏故事，端看他如何装点这副骨架

		- 不要把别人提供的摘要当做唯一的说明

			- 一本书的整体精神可以有各种不同的阐释，没有哪一种一定对

	- 将书中重要篇章列举出来，说明它们如何按照顺序组成一个整体架构

		- 一本书是一个集合体，不清楚这些构成部分，就没法理解全书

		- 一本好书，每个部分都要很有秩序的排列起来。
每个部分都要有一定的独立性
但是一定要跟其他部分连接起来

		- 如何做

			- 每一部分谈的是什么

			- 每一部分的主要部分的段落说的是什么

			- 每一部分的每个阶段有哪些重点

		- 作者拟定纲要，已写出一本好书
读者拟定纲要，已读的明白

	- 阅读与写作

		- 写作与阅读是一体两面的事

		- 读者要发现书中隐藏的骨架
作者以制造骨架为开始，想办法把骨架隐藏起来

		- 血肉与骨架一样重要

			- 血肉为全书增加必要的空间和深度

			- 一个作品应该有整体感，清楚明白，前后连贯

			- 传达知识的书每个部分可以是独立的
		小说或文学作品却不能

				- 小说是一个整体 ，所有的概念都是一个整体的概念，不能说读了一点就懂了整体概念

				- 传达知识的书仔细读一部分，就能得到整体概念

	- 发现作者意图

		- 找出作者要问的问题

		- 一本书开始写作时，都是有一个问题或一连串问题

- 如何做

	- 1.依照书本种类与主题分类

	- 2.用最简短的句子说出整本书在谈什么

	- 3.按照顺序关系，列出全书的重要部分，将全书纲要拟出后，再将各个部分的纲要一一列出

	- 4.做出作者在问的问题，或作者想解决的问题

	- 一次完成所有阶段，就会知道整本书谈的是什么

### 第二阶段

- 与作者找出共通的词义

	- 词义是可供沟通的知识的基本要素

	- 与作者达成共识是使用文字的技巧，达到沟通知识的目的

		- 找出重要单字，透过它们与作者达成共识

			- 1.处理语言的问题

				- 单字的意思

			- 2.处理语言背后的思想含义

		- 当你在传递、接受知识时，要尽可能巧妙地运用语言的技巧

			- 作者的文字要与读者相呼应。如果读者没有呼应，双方就不能达成沟通

	- 一个单字可能代表许多不同的词义，而一个词义可以用许多不同的单字来解释

	- 找出关键字

		- 看哪些字要比其他的字更重要一些

		- 一般最重要的字就是那些让你头痛的字

		- 专门用语、特殊词汇

			- 每个知识领域都有独特的专门用语

				- 只要熟悉一下那个领域，就能找出那些专门的单字

				- 只要看到不是平常惯见的字，就会知道这些字是专门用语

				- 一些日常字也会被当做专门用语

			- 特殊词汇

				- 作者与其他作者争执的某个用语就是重要的字

				- 一些字会用很特殊的方法来用
	一些字会依照这个领域的传统的方法来用

				- 其他字义不明的字，对你也很重要

			- 要努力去注意文字，找出他们传递的意义

				- 不了解这些字就一定会给你带来麻烦

				- 无法进一步了解这本书

				- 不了解这本书，对你就没有启发

				- 没有主动的阅读就会对字句毫不用心

	- 找出字义

		- 要判断关键字是否有多重意义
这些意义之间关系如何
在某个地方出现时，是其中的那个意义

		- 一定要利用上下文自己已经了解的字句，来推断不了解的字的字义

			- 唯一方法

		- 如果书中每个字都陌生，那么就不会有任何进展

	- 与作者找出共通的词义，对一本书的理解力就会大大增加

- 判断作者的主旨

	- 主旨是一种声明，表明作者对某件事的判断。

		- 是一种知识的声明，而不是意图的声明

		- 如果没有理论的支持，就是在抒发个人想法

		- 我们要了解为什么他认为我们该被说服，接受他的观点

		- 诠释作者的意图

	- 句子与主旨

		- 句子和段落是文法的单位
主旨与论述是思想与知识的单位

		- 句子表达疑问与论述，主旨则是疑问的答案和论述的总结

		- 如何找

			- 将一本书中最重要的句子圈出来，找出其中的主旨

				- 真正的关键句只有少数几句话

					- 一些需要花时间努力诠释的句子

					- 在整个论述中，阐述作者判断的部分

					- 沟通的核心是他所下的判断，与他为什么会这么判断的理由

					- 读这部分要特别仔细

				- 阅读的一部分本质就是被困惑，而且知道自己被困惑。怀疑是智慧的开始

					- 感到困扰的句子要慢慢阅读

			- 从相关文句的关联中，设法架构出一本书的基本论述

				- 找出每个句子包含的主旨

					- 用自己的话说出来

						- 无法放下作者使用的字句，那他所传达的只是那些字，而不是他的思想。他想沟通的是知识而不是文字

				- 找出书中说明重要论述的段落，如果没有，就自己去架构出来

					- 从几段中挑选出句子，整理出前后顺序的主旨以及组成的论述

					- 只要让大脑不断活动，能跟作者达成共识，找到他的主旨，就能看出他的论述是什么

	- 找出解答

		- 找出作者在书中想要解决的主要问题

			- 那些问题被解决了
为了解决问题，提出了什么新问题
那些问题没被解决

- 如何做

	- 5.诠释作者使用的关键字，与作者达成共识

	- 6.从最重要的句子中抓出作者的重要主旨

	- 7.找出作者的论述，重新架构这些论述的前因后果，以明白作者的主张

	- 8.确定作者已经解决了哪些问题，还有那些是未解决的。在未解决的问题中，确定哪些是作者认为自己无法解决的问题

## 阅读的最终目标

### 主题阅读

- 对一个特定问题，所牵涉的绝不是一本书能说清楚的

	- 准备一个书目，检索那些书与主题有关

	- 就主题建立起清楚地概念

- 如何做

	- 1.找到相关章节

		- 主题阅读中，关心的主题才是重点，而不是阅读的书

	- 2.带引作者与你达成共识

		- 把作者的词汇转成自己的词汇

	- 3.理清问题

		- 建立一组不偏不倚的主旨

	- 4.界定议题

		- 每个人对同一主题会有不同的论断

	- 5.分析讨论

		- 这本书在说什么？如何说明的？与我何干？

- 主题阅读不是要给问题提供最终答案

	- 要面面俱到，自己并不预设立场
